# CEFR curriculum path — version note (v1)

**Slug**: `cefr-curriculum-path`  
**Date**: 2026-03-25

Initial scope captured for iteration:

- Explicit **active study level** (A2, B1, …) aligned with onboarding profile but changeable in settings.
- **Structured curriculum** per level: ordered units → ordered lessons (matches JSON manifest + units or future CMS).
- **Suggested learning plan**: daily target (time or lesson count), ordered “today” queue, streak-aware nudges.
- **Progress**: extend existing `lesson_progress` semantics; add unit/path completion aggregates.
- **Weak areas**: derive from quiz/exercise outcomes (topic, grammar tag, skill); surface “Practice again”.
- **Revision sessions**: short mixed exercises drawn from completed lesson content (not a separate product).

Superseded by finalized overview + impact assessment after review/audit.
