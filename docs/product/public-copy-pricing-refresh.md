## Pricing model messaging

Pricing ladder is consistently framed as:

1. **FREE** - try the system
2. **CORE** - main plan for real A2 progress
3. **PREMIUM** - full exam-performance system

`FREE` is displayed as `FREE` (never `€0`).

## Plan positioning applied

- **Free**
  - entry/acquisition
  - limited access
  - explore fit before commitment
- **Core**
  - commercial center of gravity
  - most popular
  - strong structure + progress
- **Premium**
  - full exam prep + timed performance
  - mock exams + deep feedback + readiness depth

## Copy principles used

- learner-first, concrete, outcome-oriented
- short scan-friendly blocks
- no internal/product-builder wording
- clear upgrade logic:
  - Free -> Core -> Premium
