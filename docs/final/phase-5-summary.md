# Phase 5 Summary: User Workflows & Product Journeys

## What Changed

- **v1** included all major journeys; review suggested explicit mention of listening and daily reflection in the daily loop. Added §5.5 "Related Journeys (Same Loop)" for listening (FD-05) and daily reflection (FD-07). v1 approved for finalization.

## Major Design Decisions

1. **Single onboarding flow**: Sign-up → profile → level/goals → consent → first recommendation (FD-01).
2. **Daily loop**: Home as hub; user chooses lesson, scenario, voice, listening, or exam prep; caps and upsell at decision points (FD-12).
3. **Conversion**: Upsell at cap or CTA → trial start → use → payment or revert; BFR-013 events.
4. **Settings**: Profile edit, consent toggles, subscription management, export/delete (BFR-008, BFR-009).
